## Notas importantes:

1. **Composer** é ideal para integrações SaaS-to-SaaS simples sem código[](https://docs.mulesoft.com/composer/ms_composer_overview)​
    
2. **RPA** é usado quando não há API disponível e precisa interagir com UI legada[](https://docs.mulesoft.com/rpa-home/)​
    
3. **Flow** é a base da automação no Salesforce, com várias opções de trigger[](https://help.salesforce.com/s/articleView?id=005298960&type=1)​
    
4. **Flow Orchestrator** coordena processos complexos multi-usuário e multi-step[](https://help.salesforce.com/s/articleView?id=platform.orchestrator_flow_orchestrator.htm&type=5)​
    
5. **Anypoint Platform** fornece APIs composable para integrações enterprise[](https://concentrus.com/mulesoft-anypoint-platform-documentation/)​
    
6. **Exchange** centraliza e reutiliza assets em toda a organização[](https://docs.mulesoft.com/exchange/exchange-api)​
    
7. **Monitoring** garante visibilidade e troubleshooting de toda a solução[](https://docs.mulesoft.com/monitoring/)​
    

Todas essas ferramentas trabalham juntas para formar uma solução completa de hyperautomation, integrando dados, processos e usuários através da plataforma unificada.[](https://help.salesforce.com/s/articleView?id=005298960&type=1)​
