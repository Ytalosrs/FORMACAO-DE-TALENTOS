# 📚 Geral Informativo e Anexos

**O que é esta pasta?**
Esta é a biblioteca central de recursos administrativos, templates e guias da equipe Formação de Talentoss.

**O que você encontra aqui:**
- **📋 Modelos de Documentos:** Templates padronizados (como o `MODELO DE DAILY.md`, `pull_request_template.md`)
- **📖 Guias e Tutoriais:** Documentação de apoio (guia Git, onboarding)
- **🗺️ Mapa de Conhecimento:** Índice central de todos os documentos do repositório
- **📎 Anexos:** Arquivos de suporte que não se encaixam em projetos específicos

## 🛠️ Como Utilizar este Repositório

### 👶 Para Novos Integrantes:
1. Comece lendo o guia de [Onboarding](./ONBOARDING_NOVOS_COLABORADORES.md)
2. Configure sua pasta na seção `4 - TAREFAS`
3. Estude o [Mapa de Conhecimento](./MAPA_DE_CONHECIMENTO.md) para entender a estrutura

### 🔄 Para Rotina Diária:
- Utilize o [Modelo de Daily](./MODELO%20DE%20DAILY.md) para documentar seu progresso
- Consulte o guia de comandos Git para facilitar o versionamento

### 📚 Para Consulta Rápida:
- O [Mapa de Conhecimento](./MAPA_DE_CONHECIMENTO.md) serve como o índice central
- Os templates garantem padronização em toda a equipe
