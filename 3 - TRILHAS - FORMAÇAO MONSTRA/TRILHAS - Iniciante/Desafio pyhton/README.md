### Documentação de Desafio
Nome do desafio
## LoanShelf – Sistema de Controle de Empréstimos

Linguagem utilizada
Python

### Objetivo

Desenvolver um sistema para gerenciar o empréstimo de itens, como livros, jogos e ferramentas, permitindo o cadastro de usuários e itens, realização de empréstimos, devoluções e consultas, aplicando conceitos fundamentais de programação.

### Descrição

O desafio consistiu em criar uma aplicação em Python capaz de controlar o ciclo completo de empréstimos. Durante o desenvolvimento foram implementadas validações para garantir a integridade dos dados e o correto funcionamento das operações.

 ### Funcionalidades 
- Cadastro de itens
- Cadastro de usuários
- Registro de empréstimos
- Registro de devoluções
- Consulta de itens cadastrados
- Consulta de usuários
- Histórico de empréstimos
- Cálculo automático da disponibilidade dos itens
- Validação de dados de entrada
- Controle do limite de empréstimos por usuário

- Funções
- Listas

- Validação de entradas
- Organização do código em módulos e funções
- Lógica de programação

 ### Principais desafios encontrados

Durante o desenvolvimento, alguns desafios precisaram ser resolvidos:

- Implementar o cálculo automático da disponibilidade dos itens.
- Validar corretamente os dados informados pelo usuário.
- Garantir que um usuário não pudesse emprestar o mesmo item duas vezes sem realizar a devolução.
- Controlar o limite máximo de empréstimos por usuário.
- Manter os identificadores únicos para itens, usuários e empréstimos.
- Organizar o código para facilitar futuras manutenções.

### soluçoes adotadas

Foram criadas funções específicas para cada operação do sistema, separando as responsabilidades de cadastro, empréstimo, devolução e consulta. Também foram implementadas validações antes da execução de cada ação, garantindo a consistência das informações armazenadas.

### resultados

Ao final do desafio foi possível desenvolver um sistema funcional de controle de empréstimos, consolidando conhecimentos em Python, lógica de programação, estruturas de dados e boas práticas de desenvolvimento.
