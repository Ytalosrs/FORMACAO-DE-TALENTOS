# 🤖 Repositório de Recursos sobre Inteligência Artificial Agêntica (Agentic AI)

Bem-vindo ao repositório central de estudos, arquiteturas e práticas de desenvolvimento voltadas para a **Inteligência Artificial Agêntica**.

Enquanto a IA generativa tradicional atua como um assistente reativo que responde a prompts de forma estática, a **IA Agêntica** atua como um "chef pessoal", sendo capaz de raciocinar sobre objetivos, planejar fluxos de trabalho com múltiplas etapas, utilizar ferramentas externas (APIs, bancos de dados, web) e manter memória e estado ao longo do tempo.

Este repositório contém uma vasta coleção de livros, guias e manuais práticos que cobrem desde a infraestrutura de dados até a operacionalização de sistemas multi-agentes em produção.

## 📚 Tópicos Abordados
Os materiais deste repositório estão divididos nas seguintes áreas de conhecimento:
1. **Fundamentos e Arquitetura**: 
Explore os conceitos essenciais que diferenciam sistemas tradicionais de sistemas agênticos, com foco na infraestrutura e nos padrões de design.
- **Padrões Arquiteturais**: Compreensão de padrões como Reflection, uso de ferramentas (Tool use), planejamento e transição de tarefas (Handoff) entre agentes.
- **Agentic Mesh**: O conceito de malha agêntica para orquestrar agentes autônomos como entidades produtoras e consumidoras em ecossistemas de escala corporativa.
- **Arquitetura de Dados**: Como estruturar Data Lakes unificados, Catálogos de Dados, Bancos de Dados Vetoriais (Vector DBs) e frameworks de governança para alimentar agentes com contexto confiável.
2. **Desenvolvimento e Frameworks**: 
Recursos práticos para a construção de agentes utilizando as ferramentas mais modernas do mercado.
- **Model Context Protocol (MCP)**: O padrão universal (como um "USB-C" para IA) que permite aos agentes descobrir e invocar ferramentas e fontes de dados externas de forma padronizada.
- **Ferramentas e SDKs**: Desenvolvimento prático utilizando Claude Code, OpenAI Agents SDK, Microsoft Foundry, LangGraph, e CrewAI.
- **Codificação Agêntica**: O uso de IA para entender o contexto de repositórios inteiros (ex: via CLAUDE.md), criar planos de execução e iterar sobre o código de forma autônoma.
3. **Aplicações em Domínios Específicos**: 
Como a IA agêntica está redefinindo setores específicos:
- **Bancos e Finanças (The Agentic Bank)**: Otimização de liquidez, previsão de fluxo de caixa e monitoramento de transações em tempo real usando Agentic RAG e infraestrutura orientada a eventos.
- **Cibersegurança Ofensiva**: Automação de tarefas de reconhecimento, priorização de vulnerabilidades e testes de segmentação de redes usando agentes.
- **Compras (Procurement)**: Uso de agentes para negociar contratos, gerenciar a cadeia de suprimentos e analisar dados de fornecedores.
- **Engenharia de Dados**: Agentes capazes de monitorar a qualidade dos dados, realizar operações ETL, sugerir otimizações em data warehouses e recuperar informações através de Text-to-SQL.
4. **Operacionalização e DevSecOps (AgentOps)**: 
A transição da experimentação para a produção.
- **Docker e Contêineres**: Como isolar a execução de código gerado por IA (sandboxing), construir MCP Gateways locais e implantar agentes usando Docker Compose e Kubernetes.
- **Governança e Segurança**: Criação de barreiras de proteção (guardrails), auditorias de decisões das IAs, controle de acesso e proteção de dados sensíveis (PII).
